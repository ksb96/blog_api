# Blog_App - REST API

RESTApi's for 'Blog App' created.
Basic operations like CRUD(Create, Read, Update & Delete) can be performed. Also, 'custom exception' handling has been done.

# API categories -
1) Authentication API.
2) Category API.
3) Comment API.
4) Post API.
5) User API.

# Tables name -
1) Categories,
2) Comments,
3) Posts,
4) Role,
5) User role,
6) Users.
